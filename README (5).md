# Banking System

## Overview
This project is a simple **Banking System** written in C++. It simulates basic banking operations such as account creation, deposit, withdrawal, and balance inquiry.

## Features
- Create new bank accounts
- Deposit and withdraw money
- View account details
- Manage multiple users
- Simple console-based interface

## Requirements
- C++ compiler (e.g., GCC or Visual Studio)
- Basic knowledge of C++ programming

## How to Compile and Run
1. Open terminal or command prompt.
2. Navigate to the project directory.
3. Compile the file using:
   ```bash
   g++ "Banking System.cpp" -o BankingSystem
   ```
4. Run the executable:
   ```bash
   ./BankingSystem
   ```

## File Structure
- `Banking System.cpp`: Main C++ source code file containing all banking system logic.
